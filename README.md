### 🚀 Featured Engineering

#### 🖨️ SPOOL: 3D Printing Inventory Manager
*An offline-first Android utility to seamlessly track filament inventory, weights, and colors.*

> **Architecture:** Built entirely with a local-first Room database for zero-latency data access.<br>
> **UI & Routing:** Fluid Jetpack Compose interface with compile-time safe routing utilizing the Navigation 3 library alongside `NavDisplay`.

#### 🎣 Bislei: Community Social Platform
*A location-based social app for discovering fishing spots and sharing real-time hotspots.*

> **Architecture:** Utilizes Firebase Auth and Firestore for real-time synchronization. Data is strategically fetched from Firestore and sent to the local database for optimized caching.<br>
> **Integration:** Maps SDK integration for precise location tracking and discovery.
