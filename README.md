## 🚀 Featured Engineering

### [🖨️ SPOOL: 3D Printing Inventory Manager](https://play.google.com/store/apps/details?id=com.aadil.spool)
> **Status:** Production (100+ Active Downloads)<br>
> **Summary:** A utility designed to seamlessly track filament inventory, weights, and colors with zero latency.

* **The Architecture:** Engineered as an offline-first application utilizing a local Room database as the single source of truth. 
* **UI & Routing:** Built entirely in Jetpack Compose with compile-time safe, type-safe routing utilizing the Navigation 3 library alongside `NavDisplay`.

<br/>

### 🎣 Bislei: Community Social Platform
> **Summary:** A location-based social application for discovering fishing spots and sharing real-time hotspots.

* **Data Synchronization:** Implemented a robust data layer using Firebase Auth and Firestore for real-time remote syncing.
* **Caching Strategy:** Strategically engineered to fetch remote data from Firestore and persist it to a local Room database, optimizing network calls and enabling rapid local caching.
* **Integrations:** Deeply integrated the Google Maps SDK for precise location tracking, clustering, and discovery.
